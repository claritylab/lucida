\subsection{Detailed Description}
Bigram probs and bo-\/wts, and trigram probs are kept in separate tables rather than within the bigram\-\_\-t and trigram\-\_\-t structures. 

These tables hold unique prob and bo-\/wt values. The following tree structure is used to construct these tables of unique values. Whenever a new value is read from the L\-M file, the sorted tree structure is searched to see if the value already exists, and inserted if not found.
